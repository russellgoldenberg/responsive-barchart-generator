Responsive Bar Chart Generator v0.2

A complete HTML/CSS solution for quick & elegant responsive bar charts. Code is unminified for easy customization.

Copy and paste data from Excel or Google sheet. Data should be name and value pairs. Use numbers in the format you want them displayed (ex. if dollars, then use $4 or $4.00, not 4). Load sample data.

Options

Select options, replace underlined text with your copy below, then click 'generate code'. If you don't want a hed or a subhed, delete the contents. The generated code will remove the whitespace.

Responsive Bar Chart Generator v0.1

A complete HTML/CSS solution for quick & elegant responsive bar charts. Code is unminified to easily customize.

Copy and paste data from Excel or Google sheet. Data should be name and value pairs. Use numbers in the format you want them to be displayed (ex. if dollars, then use $24 or $24.00, not 24). Demo
